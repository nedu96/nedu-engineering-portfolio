# Nedu Anandarajan — Software Engineering Portfolio

A portfolio focused on real-time systems, maritime technology, industrial IoT,
backend engineering and distributed platforms.

Live site: [nedu-engineering-portfolio.nedu1996.chatgpt.site](https://nedu-engineering-portfolio.nedu1996.chatgpt.site)

## Portfolio modes

- **Command Centre** — a dark maritime systems interface inspired by radar,
  telemetry and operational software.
- **Editorial Engineer** — a warm editorial layout inspired by engineering
  notebooks, signal plots and system diagrams.

Visitors can switch between both visual modes without leaving the page.

## Featured experience

- Real-time Java and Python services for maritime docking and mooring systems
- Modbus TCP/RTU, TCP socket, serial and custom protocol integrations
- AIS/NMEA 0183 decoding and IVEF vessel-target exchange
- PostgreSQL, TimescaleDB, RabbitMQ, WebSockets and Docker
- AWS, Prometheus, Grafana and production troubleshooting
- ROS programming for Fetch and Pepper robots
- Gazebo virtual environments created from spatial data, including robot
  spawning and simulation workflows
- Raspberry Pi coffee-machine telemetry published to a Django dashboard

## Development

```bash
npm install
npm run dev
```

The site is built with Next.js, React, TypeScript and CSS. It is responsive,
keyboard accessible and respects reduced-motion preferences.
